# Random Facts Generator

A simple, interactive web application that displays random computer-related facts with a clean, modern interface and smooth transitions.

## Features

- Single-page application
- Clean, modern UI design
- Smooth fade transitions when displaying new facts
- Responsive design that works on both desktop and mobile devices
- Collection of interesting computer-related facts
- Simple one-click interaction

## Setup

1. Download the `index.html` file
2. Open it in any modern web browser
   
That's it! No additional setup or dependencies required.

## Usage

1. Open the application in your web browser
2. Click the "Generate Fact" button to display a new random fact
3. Each fact will fade in with a smooth transition effect

## Technical Details

Built using:
- HTML5
- CSS3
- Vanilla JavaScript

- `index.html` (contains all HTML, CSS, and JavaScript code)
